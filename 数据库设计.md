### 图书表 books

| 字段名  | 字段类型 | 备注               |
| ------- | -------- | ------------------ |
| id       | int      | 自增               |
| book_id  | varchar  | UUID生成           |
| name     | varchar  |                    |
| author   | varchar  |                    |
| price | decimal  |                    |
| publish  | date     | 出版时间           |
| reserve  | int      | 库存，被同步锁保护 |

### 用户表 users

| 字段名    | 字段类型 | 备注     |
| --------- | -------- | -------- |
| id        | int      | 自增     |
| user_id   | varchar  | UUID生成 |
| user_name | varchar  |          |
